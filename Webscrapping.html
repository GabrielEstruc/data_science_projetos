Projeto para extrair os destaques de notcias do veculo de mdia BBC

  • Utilizando o BeutifulSoup pude aplicar a tcnica de webscrapping para extrair informaes das headlines do dia.
  • Tendo as informaes tratadas, utilizei o WorldCloud para extrair e plotar em palavras-chave o destaques atualizados.
  • Com isso podemos ter acesso a informaes de forma veloz toda vez que rodarmos o script.

for i in headline: #utilizando um lao para extrair e concatenar todas as headlines
    text+=" "+i.text+" "
